# Third-Party / Supplier Onboarding - RACI Matrix

Security due diligence across the supplier lifecycle (ISO 27001 A.5.19-A.5.22).

**Legend:** R = Responsible, A = Accountable, C = Consulted, I = Informed

| Activity | Business Sponsor | Procurement | Security / TPRM | Legal | Data Owner | CISO |
| --- | --- | --- | --- | --- | --- | --- |
| Initiate vendor request | R | A | C | I | C | I |
| Tier vendor by risk | C | C | R | I | C | A |
| Run security due diligence | I | C | R | C | C | A |
| Negotiate security & DPA terms | C | R | C | A | C | I |
| Approve onboarding | C | C | R | C | C | A |
| Periodic reassessment | C | C | R | I | C | A |
| Offboard & revoke access | R | C | A | I | C | I |
