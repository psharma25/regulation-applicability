Run this weekly on autopilot

This page enriches data locally on each run (estimates, dates, scores, staleness). For a live weekly web pull, drive it from a scheduler — matching the single-file + Ollama + GitHub pattern:

name: weekly-mf-scan
on:
  schedule: [{ cron: '0 13 * * 1' }]   # Mondays 9am ET
  workflow_dispatch:
jobs:
  scan:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - run: python scan.py --towns towns.txt --out data.json
      # scan.py: pull MLS/LoopNet/Foreclosure.com per town,
      # enrich with local Ollama (llama3.1) for value-add scoring,
      # deterministic math for est. price / cap — never the LLM.
      - run: |
          git add data.json && git commit -m "weekly scan" && git push

scan.py writes data.json; this page fetches it on load. Pricing & cap math stay deterministic; the LLM only classifies text (value-add, condition cues).

Data provenance & what the scores mean

Asking vs Est.
ASK = real list price from the listing. EST = modeled estimate when no price is published: town median × a unit multiplier (Boston priced per-unit). A transparent estimate, not an appraisal.
Listed date
Derived from days-on-market where the listing disclosed it; blank when unknown. The agent recomputes on each run.
Sold comps
Sold price & date are public record but not pre-filled — pulled per-property from Redfin/Zillow sold pages or the registry. Sold rows are tagged and link to the sold source.
Value-add
Auto-derived from listing language (below-market rents, vacant units, needs reno, as-is, upside). Verify on site.
Main road
Inferred from the street name (Main, Boston Post, Turnpike, Rte, Speen, Worcester Rd, major aves). A heuristic — confirm.
Walk score
Real Walk Score where verified (Framingham 42, Marlborough 41); otherwise blank with a walkscore.com lookup. Not invented.
Wet basement / Sunlight
Inspection-only. Never pre-filled — shown as “?” until you assess on site or from disclosure/photos. Sunlight flips to “good” only if the listing explicitly says sun-filled/bright.
Owner / phone
Not pre-filled — from the town/Boston assessor (free) and a compliant skip-trace. Contact column carries the public listing agent where named.
Compliance: phone outreach is governed by TCPA + the National/MA Do-Not-Call registries; foreclosure-owner contact is further restricted under MA Ch. 244. Scrub and log consent. Everything here stays in your browser.
