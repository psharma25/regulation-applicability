Open Source Stack

Local AI Architecture

Runs entirely on your infrastructure. No patient data, product data, or company information is sent to external services.

INGESTION
Regulation Harvest
Scrapy + Apache Tika for PDF/HTML extraction. LangChain document loaders for structured sources. Daily cron with hash-diff change detection.
VECTOR STORE
ChromaDB / pgvector
Local vector database. Embeddings via nomic-embed-text (Ollama). Chunk size 512 tokens with 64-token overlap. HNSW index for fast retrieval.
LOCAL LLM
Ollama + Llama 3 / Mistral
Ollama serves models locally. Llama 3.1 8B for query answering, Mistral 7B for structured extraction. Runs on CPU or consumer GPU.
AGENTIC LAYER
LangGraph Agents
Applicability agent, gap analysis agent, roadmap generator agent. Each uses RAG retrieval + structured output. Human-in-loop approval gates.
BACKEND
FastAPI + PostgreSQL
REST API for profile storage and analysis results. PostgreSQL with pgvector extension. Docker Compose deployment for full local stack.
MONITORING
Regulatory Change Alerts
Automated diff detection triggers re-analysis when regulations change. Email or Slack alerts to compliance owners with impact summary per product.
