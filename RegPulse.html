How To Use RegPulse

Five capabilities, one workflow.

1 · Find what applies
In Applicability, pick a company/product or example, or enter your own, then run it to see what applies and why.
2 · Read each regulation + source
In Regulations, open any card for the summary, the official source link, and a roadmap download (HTML/Excel).
3 · Run a scan yourself
In Live Scan, pick a scope and run the pipeline on demand — it re-reads every source, fingerprints it, and reports what changed, with a downloadable report.
4 · Track weekly change
In Weekly Delta, review new / updated / unchanged regulations and download the report.
5 · Build the roadmap
After applicability, download a single combined, de-duplicated roadmap across all applicable regulations.
6 · Get the controls
In Controls, download one control or the full library in HTML or Excel.
Automate it
This file is self-contained. To run the scan weekly without you, push the repo to GitHub — the included Actions workflow runs it every Monday at $0. See the README for the cron alternative.
